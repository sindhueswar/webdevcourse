Front-end web development

Front-end web development is the development of the graphical user interface of a website,
through the use of HTML, CSS, and JavaScript,
so that users can view and interact with that website.

HTML5

The HTML5 syntax is no longer based on SGML despite the similarity of its markup.
It has, however, been designed to be backward-compatible with common parsing of older versions of HTML
